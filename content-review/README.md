# Guide de Révision du Contenu

Ce dossier contient l'intégralité du contenu textuel de la plateforme **Je suis au Cameroun**, organisé par sections pour faciliter votre lecture et vos corrections.

## Comment utiliser ces documents ?

1.  **Lecture :** Parcourez chaque fichier `.md` (Markdown). Ils sont lisibles comme du texte simple.
2.  **Correction :** Si vous souhaitez modifier un texte, vous pouvez :
    *   Modifier directement les fichiers si vous êtes à l'aise avec le format.
    *   Ou copier-coller le texte dans un document (Word/Email) avec vos corrections.
3.  **Rapport :** Une fois vos corrections terminées, renvoyez-nous les fichiers ou une liste de changements.

## Structure des fichiers

*   `01_Landing_Page.md` : Page d'accueil complète (Hero, Pourquoi, Défis, FAQ).
*   `02_A_Propos.md` : Vision, écosystème et mot du Président.
*   `03_Comment_Ca_Marche.md` : Guide étape par étape pour les utilisateurs.
*   `04_Contact_Et_Support.md` : Informations de contact et formulaire.
*   `05_Mentions_Legales_Et_Confidentialite.md` : Textes juridiques et RGPD.
*   `06_Formulaires_Soumission.md` : Tous les champs des formulaires de profil et projet.
*   `07_Dashboards_Admin_Labels.md` : Menus et boutons des espaces réservés.
*   `08_Navbar_Et_Footer_Labels.md` : Liens de navigation communs.
*   `09_Contenu_Dynamique_Exemples.md` : Exemples de données provenant de la base de données.

---
*Note : Pour les corrections massives de données (Missions, Actualités, Organisations), vous pouvez également utiliser l'outil d'export/import Excel disponible dans votre espace Superadmin.*
